## 新 YDoc v4.0.0 版本发布啦！ 🎉🎉🎉

### v4.0.0 (2018.03.29)
- 重构: 经过长达数年的文档构建工作，我们积攒了大量构建文档的经验，此刻我们卸下历史包袱、厚积薄发、精雕细琢地打造了拥有卓越的视觉与交互体验的文档构建工具！
- 灵活的插件机制: 万物皆插件，YDoc 虽然是一个非常轻量的工具，但插件机制让它灵活地应用于各种复杂场景
- 响应式设计，支持手持设备访问
- 简洁而严谨的 UI 设计
- 文档设计规范

### v4.0.3
- 新增系统组件 Icon，可自定义网站 icon
- 新增 `nav` 钩子
- 支持 nodejs 通过 require('ydoc')(config) 生成文档
- 修复 jsx 高亮
- 修复编码问题
